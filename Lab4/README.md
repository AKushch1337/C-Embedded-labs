# Lab4: String task

> My variant is 10
> My task: Count the number of digits in a string

## Build the project

```
git clone https://github.com/AKushch1337/C-Embedded-labs
```

```
cd /path/to/project/C-Embedded-labs/Lab4
```

### Using Makefile

```
make
```

### Using gcc(if you have it installed)

```
gcc main.c -o lab4
```

## Run the project

```
./lab4.exe
```

## Clean the .exe

```
make clean
```
