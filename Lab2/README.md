## Build the project

```
git clone https://github.com/AKushch1337/C-Embedded-labs
```


### Using Makefile

```
make
```

### Using gcc(if you have it installed)

```
gcc main.c -o lab1
```

## Run the project

```
./lab1.exe
```

## Clean the .exe

```
make clean
```
